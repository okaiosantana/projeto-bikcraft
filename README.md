### Animais Fantásticos

Projeto front-end desenvolvido com **JavaScript puro (ES6+)**, com foco em **interatividade, performance e experiência do usuário**.
O site simula uma aplicação institucional sobre animais, explorando diversos padrões modernos de UI/UX e integração com APIs externas.

Este projeto foi criado como parte de um curso avançado de JavaScript e está sendo utilizado como **peça de portfólio profissional**.

---

### Funcionalidades Implementadas

* Navegação interna com scroll suave, melhorando a fluidez da experiência do usuário.
* Menu com sub-menus dropdown, controlados via JavaScript.
* Animação de seções ao scroll, ativadas conforme o usuário navega pela página.
* Otimização de performance com debounce aplicado aos eventos de scroll.
* Seção FAQ interativa (Accordion List), permitindo expandir e recolher conteúdos.
* Navegação por abas (Tab Navigation) com imagens clicáveis, exibindo informações dinâmicas de cada animal.
* Modal de login, controlado via JavaScript.
* Tooltip personalizada na seção de contato, exibida ao interagir com o mapa.
* Animação dinâmica de números, exibindo dados estatísticos de forma visual.
* Consumo de API interna para carregar dinamicamente os dados da seção “Números”.
* Integração com API externa para obter a cotação atual do Bitcoin em BRL e calcular valores na seção “Doações”.
* Sistema de horário de funcionamento, que identifica automaticamente se a loja está aberta com base no horário local do usuário.
* Layout totalmente responsivo, garantindo boa usabilidade em dispositivos móveis.

---

### Tecnologias e Conceitos Utilizados

* JavaScript (ES6+)
* Manipulação do DOM
* Eventos e delegação de eventos
* Modules (ES Modules)
* Fetch API e consumo de APIs externas
* Animações baseadas em scroll
* Debounce para otimização de performance
* Boas práticas de organização e reutilização de código
* UX/UI focado em interatividade

---

### Objetivo do Projeto

Demonstrar conhecimento prático em **JavaScript moderno**, criação de interfaces interativas, integração com APIs e aplicação de conceitos de **performance e experiência do usuário**, servindo como projeto de portfólio para oportunidades profissionais na área de desenvolvimento front-end.
